# OPD Closing Duties Checklist

**Complete all tasks before end of shift**

---

## 1. Check Undispensed Orders

### Timeline
- [ ] Complete **1-2 hours before OPD closing** to allow time for pickup/redelivery

### Actions
- [ ] Call customers who did not pick up orders
- [ ] Call customers whose orders were returned by delivery driver
- [ ] Ask if they want to: pick up, have redelivered, or cancel

---

## 2. Handle Pickup Orders

### Customer Already Received Order
- [ ] Dispense the order in **GIF Dispense** to finalize

### Customer Wants to Cancel
- [ ] Direct customer to cancel through app/website in **"My Account"** → Purchase History
- [ ] OR direct to call **Customer Care at 1-800-Walmart**
- [ ] If customer cannot cancel in "My Account" or requests you cancel:
  - [ ] Contact **Store Support** to cancel the order

### Customer Wants to Pick Up
- [ ] Leave order in staging area
- [ ] When customer arrives, dispense order in **GIF Dispense**

### Important
- [ ] **Cancel same-day if not picked up**

---

## 3. Handle Delivery Orders

### Customer Wants Re-delivery
- [ ] Request new driver through **GIF Dispense** → **Orders Needing Attention**

### Customer Wants to Cancel
- [ ] Cancel through **GIF Dispense** → **Orders Needing Attention**

### Customer Wants to Pick Up (Scheduled Delivery)
- [ ] Leave order in staging area
- [ ] Convert order to pickup in **GIF MyStore** → **Dispense Exceptions tile**
- [ ] When customer arrives, dispense through **GIF MyStore** → **Orders Needing Attention**

### Important
- [ ] **Cancel same-day if not picked up**

---

## 4. Process Returns

### Timing
- [ ] Complete during downtime
- [ ] **Must be completed by end of day**

### Actions
- [ ] Return refunded items to modular home
- [ ] Take damaged items (with order numbers) to claims for processing

---

## 5. Clean and Prepare Area

### Deadline
- [ ] **Complete by 10 PM** to allow next day's associates to begin immediately

### Backroom Crew Board
- [ ] Clean board to be ready for next day

### Equipment Organization
- [ ] Stack and organize dollies and handles in designated area
- [ ] Stack all empty totes and containers
- [ ] Maintain equipment close to brand new state

### Tote Management
- [ ] Ensure totes remain clean
- [ ] Remove visibly dirty totes from use
- [ ] Set aside dirty totes for cleaning
- [ ] Clean totes using either:
  - Clean in Place Procedure, OR
  - Three-compartment sink (produce or bakery department)
- [ ] Remove all tote labels and label residue

### Pick Cart Preparation
- [ ] Stock pick carts with supplies:
  - [ ] Bags
  - [ ] Labels
  - [ ] Produce bags
  - [ ] Meat bags
  - [ ] Reusable bags

### Final Cleaning
- [ ] Clean floors
- [ ] Remove all garbage

---

## 6. Secure Resources

### Locked Cabinet
- [ ] Place batteries for handhelds on chargers
- [ ] Place batteries for portable printers on chargers

### Pickup Cell Phone
- [ ] Clear all messages and texts
- [ ] Ensure **no customer contact data** is saved
- [ ] Check voicemail is up to date with relevant messaging
- [ ] Place on charger

### Final Security
- [ ] Lock storage cabinet
- [ ] Lock filing cabinet
- [ ] Lock safe
- [ ] Place all keys in lockbox

---

- [ ] All equipment secured
- [ ] All customer issues resolved
